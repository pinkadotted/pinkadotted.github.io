Android App

As part of a course project, my team and I identified the persistent problem of laundry unavailability in the campus hostel. We used Java and Android Studio to develop an application to solve this problem. We put to use the concepts we had learnt in the course, such as good software Design principles, as well as Object oriented programming. WashWatch, bridges the frustrating gap in the current hostel laundry system by allowing residents to obtain real-time laundry machine statuses at the click of a button, wherever they are.
